Recover your Microsoft Exchange mail and mailboxes in minutes by using Dell AppAssure backup, recovery and replication software. Your mail and mailboxes can be back online within minutes, because AppAssure lets you prioritize access to your most-needed data while restoring the whole database in the background.

How it works:

Block level snapshots
AppAssure captures your Exchange data, as well as the application itself, in a collection of block-level snapshots for backup.
Application awareness
AppAssure makes sure that all open transactions and rolling transaction logs are completed and caches are flushed to a disk before it creates a snapshot. AppAssure leverages Microsoft VSS for application-consistent snapshots.
Fast restore technologies
AppAssure restoration of metadata allocation tables and database structures, along with its filter driver and queuing technology, let you get back online within seconds after a failure.
File prioritization
The AppAssure filter driver sits on the Exchange Server agent and intercepts any data request from the user. If a user attempts to access a file or email message while a restoration is in progress, the filter friver will determine whether that particular file (or email) has been restored, and prioritize the immediate restore of the requested file as necessary. Your end user gets fast access while recovery continues in the background.
